# Institutional Forex Broker Theme

## Installation
1. In WordPress Admin, go to **Appearance → Themes → Add New → Upload Theme**.
2. Upload `institutional-forex-broker.zip`.
3. Activate the theme.
4. Go to **Appearance → Editor** to edit header, footer, templates, and page sections.
5. Create pages with these slugs to use dedicated templates automatically:
   - `about-us`
   - `markets`
   - `platforms`
   - `introducing-broker`
   - `pamm-account`
   - `contact`

## Configuration Notes
- Menus are editable in the Site Editor navigation block.
- Contact form block uses a shortcode placeholder compatible with Contact Form 7.
- All homepage and section content is built with Gutenberg blocks and fully editable.
